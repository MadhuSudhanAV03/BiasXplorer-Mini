## How to get started

### Frontend

```bash
cd frontend
npm i
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
flask run --port 5000
```

## Development

- Frontend runs on: http://localhost:5173
- Backend runs on: http://localhost:5000
- Make sure both servers are running for full functionality
